## Installation

```bash
$ npm install

$ npm install -g @nestjs/cli

$ npm install --save @nestjs/typeorm typeorm mysql2

$ nest g resource tasks

$ npm i --save @nestjs/config

$ npm install class-validator

$ npm install class-transformer
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
